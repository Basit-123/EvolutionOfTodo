# The Evolution of Todo
Mastering Spec-Driven Development & Cloud Native AI

## Project Overview
This project follows a 5-phase progression from a simple CLI to a distributed cloud-native AI system.

### Phase 1: In-Memory CLI
A Python-based command-line interface for basic todo management.

#### Setup (Phase 1)
1. Ensure you have `uv` installed.
2. Install dependencies:
   ```bash
   uv pip install rich
   ```
3. Run the app:
   ```bash
   python -m apps.cli.src.main
   ```

## Evolution Path
1. **Phase I**: In-Memory Python CLI
2. **Phase II**: Full-Stack Web App (Next.js + FastAPI + Neon DB)
3. **Phase III**: AI-Powered Chatbot (OpenAI Agents SDK)
4. **Phase IV**: Local K8s Deployment
5. **Phase V**: Advanced Cloud (Kafka + Dapr)
